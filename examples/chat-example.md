POST http://localhost:8080/chat/new-message

*Request*

```json
{
    "message": "Hello, what's your name?"
}
```

*Response*

```json
{
    "reply": "I don't have a personal name. I'm an AI designed to provide information and assist with tasks, but I don't have a personal identity. You can think of me as a helpful tool or a conversational partner, though! How can I help you today?"
}
```
